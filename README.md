# To-Do List - Python Project 1
**Industrial Training | DecodeLabs | Batch 2026**

A priority-based CLI To-Do List with file persistence.
Built during industrial training at DecodeLabs.

## Features
- Add tasks with priority (high/medium/low)
- View all tasks with status
- Mark tasks as done
- Delete tasks
- Data saved to tasks.json

## Project Structure
    project1-todo-list/
        todo.py          -> main program
        tasks.json       -> auto-created when you add tasks
        README.md        -> project documentation
        .gitignore       -> files to ignore in git

## Sample Output
    === TO-DO LIST ===
      1. Add Task
      2. View Tasks
      3. Mark Done
      4. Delete Task
      5. Exit

## Tech Stack
- Python 3.x
- Libraries: json, os (built-in only)
